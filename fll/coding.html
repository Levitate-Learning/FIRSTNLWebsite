SPIKE Prime

The SPIKE Prime robots can be programmed using the LEGO Education SPIKE application. This Application can be downloaded on Windows or iPads. You can also use the website version of the SPIKE app on Chromebooks. The website version of the app does not require any downloads.

Getting Started

The LEGO SPIKE application contains many interactive units to learn the various capabilities of the SPIKE robots and how to utilize them. One of these units is getting competition ready for FIRST Lego League. This unit will guide you and your team through the fundamentals of a FIRST Lego League robot such as movement, using color sensors, and designing detachable tools.

You can access this unit by opening the unit page on the SPIKE app home page, or click the link below.
